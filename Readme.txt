# Thrium Website

Welcome to the repository for our startup website project! This project is built using HTML, CSS, and JavaScript, and it serves as the main website for our startup.

## Description

This website showcases our startup's products, services, and mission. It provides information about our company, contact details, and a platform for users to interact with us.

## Usage

The website consists of the following directories and files:

- `index.html` - The main HTML file that represents the structure of the website.
- `css/` - Directory containing CSS files for styling the website.
- `js/` - Directory containing JavaScript files for adding interactivity to the website.
- `images/` - Directory for storing image assets used in the website.

Feel free to modify the existing HTML, CSS, and JavaScript files according to your requirements. You can also add new files or directories as needed.
